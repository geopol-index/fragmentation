Description
The Geopolitical Fragmentation Index is estimated using a dynamic hierarchical factor model that treats fragmentation as a set of latent variables. Combining 16 widely used empirical indicators as noisy proxies, the model captures both the specific forces driving fragmentation—across trade, finance, mobility, and politics—and the broader trends shared across these dimensions, yielding an operational measure of geopolitical fragmentation.
